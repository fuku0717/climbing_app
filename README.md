## アプリケーション名
climbing_app


## アプリケーション概要
登山好きの人、これから登山を始めたい人が、既に投稿されている山の情報を見ることができます。

## アプリケーションを作成した背景
これから登山を始めたいけれど、どの山に登ろうか迷っている人が情報を手に入れる、あるいは登山好きの人が登った山についての情報を共有できる場を提供したいと思い、ブログ形式で情報を投稿できるアプリを作成しました。
